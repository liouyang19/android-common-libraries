# Project: android-common-libraries (Android)

## Gradle Rules
- **DO NOT** use global `gradle` command.
- **ALWAYS** use `.\gradlew.bat` for Windows environment.
- Path: `./gradlew` (Linux/Mac) or `gradlew.bat` (Windows).

## Environment
- Local SDK: Use the one defined in `local.properties`.
- Build Tool: Prefer `./gradlew assembleDebug` for testing.
